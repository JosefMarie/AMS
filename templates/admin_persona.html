Admin Pro Tips

  • → Data Isolation: Always double check that each registered trainer has a designated school name. This ensures they can only view and co-teach classes within their boundaries.
  • → Primary Color Changes: When updating the Primary Accent Color, use standardized hex format (e.g. `#4f46e5`) to avoid CSS conflicts.
  • → Live Logs: Audit trails cannot be altered or cleared from the dashboard to guarantee audit compliance.
