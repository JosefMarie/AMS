Step 1
Starting Point

Dashboard Overview

After logging in, your Dashboard is the command centre. It gives you a real-time snapshot of all your classes, students, recent marks, and quick-access shortcuts to every major tool.

📊 What you see

Total students, class count, recent activity feed, and pending co-teaching requests.

🔗 Quick Actions

Click any class card to enter the classroom view, or use the sidebar for direct navigation.

📅 Timeline

Use the Academic Year & Term selector in the top bar to view historical data.

Go to Dashboard
Step 2
Class Management Collab

My Classes & Co-Teaching

Create and manage your classrooms. Each class has its own student roster, modules, attendance records, and marks. You can also invite co-teachers from your school to collaborate on shared rosters.

Create a Class

Dashboard → "Create Class" button. Enter class name, level, and trade. Students are added from the class management page.

Add Co-Teachers

In "Manage Class" view, scroll to the School Directory. Select a colleague from the same school and click "Add Co-Teacher" directly.

Promote Students

At the end of the year, use "Promote Students" inside the class to bulk-advance students to the next level automatically.

Step 3
Gradebook New

Modules, Assessments & Gradebook

Add subject modules to your class, then create assessments (Formative, Internal, or Summative) and enter marks individually or via bulk CSV import. Use the interactive gradebook for a spreadsheet-style view.

📝 How to Add Marks

  1. Go to "Gradebook" in the sidebar
  2. Click "Create Assessment"
  3. Select module, type (FA/IA/SA), term
  4. Enter marks in the table or upload CSV

📤 Bulk Import Steps

  1. Go to Gradebook → "Bulk Import"
  2. Download the CSV template provided
  3. Fill in marks per student row
  4. Upload and validate automatically

Step 5
AI Powered ✨ New

AI Scheme of Work Generator

Generate a full-year, CBTA-compliant Scheme of Work for any curriculum module in seconds. The AI distributes all Learning Outcomes and Indicative Content intelligently across 3 terms, assigns durations, suggests learning activities, resources, and formative assessments — and lets you edit every cell before printing.

📋 Step 1 — Select

Choose the curriculum module, class name, school year, and an optional template.

🤖 Step 2 — Generate

Click "Generate with AI" and the system writes a structured week-by-week plan across all 3 terms.

✏️ Step 3 — Edit & Print

Use the full inline editor to adjust any row, then print a polished PDF version.

Step 8
Optional ✨ New

Personal Gemini API Key

If your school's shared Gemini API quota runs out, you can add your own personal Google Gemini API key directly in your profile. Once set, all your AI features — session plans, Scheme of Work generation, and quiz generation — will use your personal key automatically, without affecting other teachers.

🔑 How to Add Your Key

  1. Go to Edit Profile from the sidebar
  2. Find the Personal Gemini API Key field
  3. Paste your key (starts with AIzaSy...)
  4. Click Save Changes

🌐 How to Get a Free Key

  1. Visit aistudio.google.com
  2. Sign in with your Google account
  3. Click "Get API Key"
  4. Copy and paste it into your profile

Your key is stored securely and never shared. Priority order: Your Key → School Env Key → Global School Setting.

Go to Edit Profile

Pro Tips

  • → Always select the active term before entering marks — use the top-bar selectors.
  • → Use "AI Advisor Pro" over the standard generator for richer, more detailed plans.
  • → Co-teachers can only modify their own modules — your data stays private.
  • → Use the Resource Library to share session plan PDFs with students.
  • ✨ Add your personal Gemini key in your profile to avoid school quota limits.
  • ✨ Generate your Scheme of Work before the term starts — then edit it week by week.
